# Databases-Assignment
Databases assignment

-- Flashcard study -- 

The plan is to create a flashcard study tool to mainly help with the FE1 exams. These are the 
exams which you must pass to qualify as a solicitor in Ireland. The about page explains this better. 

The app creates flashcards and adds them to a database on the 'flashcards.html' page. The flashcards can be studied on the study page, and then quizzed on the quiz page. 

The flash cards are sorted into 8 subjects/topics, which can be broken down further into subtopics if the user wishes. 

I have filled out a few in the topic "The Law of Torts" which I'm hoping you might be able to see on your end, so that you can flick through them.
